# Digital-Nurture-Java-FSE

This repository contains the exercises, assignments, and practice programs completed during the Digital Nurture Java Full Stack Engineer training program. The repository is divided into different modules based on the technologies covered in the training.

The goal of this repository is to keep all learning materials and hands-on work organized in one place while improving problem-solving and development skills.

---

## Modules Included

### Module 1 - Front-End Development

This module contains exercises and mini projects built using:

- HTML
- CSS
- JavaScript
- Bootstrap

Topics practiced:

- Web page structure using HTML
- Styling using CSS
- JavaScript basics and DOM manipulation
- Responsive design with Bootstrap
- Simple front-end projects

---

### Module 2 - SQL

This module contains SQL queries and database-related exercises.

Topics covered:

- Basic SQL queries
- Joins
- Aggregate functions
- Subqueries
- Views
- Database operations

---

### Module 3 - Core Java

This module contains Java programs and coding practice exercises.

Topics covered:

- Java fundamentals
- Object-Oriented Programming
- Arrays and Strings
- Collections
- Exception handling
- File handling
- Multithreading
- Java 8 features
- JDBC basics
- Database connectivity using JDBC
- CRUD operations
- PreparedStatement
- ResultSet handling

---
